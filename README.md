## Amplifyapp

React notes app that allows users to create, delete, and list notes.



##### open your terminal or command prompt
```bash
# Clone the repository
```

### Build Setup
```bash
# npm install 
```

### Run react application
```bash
# npm start 
```

### Functionality
- [x] Registers a User
- [x] User login and logout
- [x] create, delete, and list notes
